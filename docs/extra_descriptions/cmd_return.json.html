This rule is handy when you are used to or would like to use the US English Layout but have one of the layouts with the big Return key . Then the Backslash (\) key next to the Return (⏎) becomes really annoying.

You can remap the backslash (\) key to return_or_enter with a Simple Modification and then enable this Complex Modification to make your backslash and pipe keys accessible when you press your right Command (⌘).
